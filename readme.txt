Необходимо включить Режим разработчика в настройках расширений chrome://extensions/
После перетащить папку или загрузить распакованное расширение.
На странице ППОЗ должна появиться кнопка "Создать инц. на ЕСРОО". При первом входе и переходе по закладкам обращения плагин не подхватывается. Для того чтобы на страницу плагин загрузился необходимо обновить страницу. До того как нажать убедитесь что открыта закладка в этом же браузере со страницей создания инцидента в ЕСРОО.
В настройках расширения есть возможность настройки шаблона сообщения. Выводимые данные берутся из json данных возвращенных по запросу ППОЗ, которые можно посмотреть в DevTools браузера Chrome в закладке Network.
Ссылка запрос http://ppoz-service-bal-01.prod.egrn:9001/manager/requests/logs/byId?id=[номер обращения]
